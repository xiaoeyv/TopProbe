# 顶尖探针
 服务器探针
帮助：

* 运行出现`ModuleNotFoundError: No module named 'psutil'`
请在命令行内输入：

```
pip install psutil
```

* 无法安装模块

1. 提示`Python.h: No such file or directory`
   可能是你没有安装`python3-devel`，请使用`yum`等程序安装后再执行命令。
